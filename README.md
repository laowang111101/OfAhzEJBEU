# 前言

欢迎来到本二手车交易系统的设计与实现项目。这是一个基于Java和MySQL开发的实战项目，适合作为计算机专业毕业生的设计分享。在这里，你将了解到项目的具体实现和关键技术的应用。本项目旨在帮助大家更好地掌握Java开发技能，提高解决实际问题的能力。

## 内容介绍

二手车交易系统是一个综合性的项目，涵盖了用户注册、登录、浏览车辆信息、发布车辆、查看个人中心等功能。本项目基于B/S架构，采用前后端分离的方式进行开发，前端负责展示用户界面，后端负责数据处理和业务逻辑。通过本项目的实战演练，你可以了解到二手车交易系统的整体架构和实现过程。

## 技术介绍

- **语言：** Java
- **使用框架：** Spring Boot
- **前端技术：** JS、Vue、CSS3
- **开发工具：** IDEA/Eclipse
- **数据库：** MySQL 5.7/8.0
- **数据库管理工具：** phpstudy/Navicat
- **JDK版本：** jdk1.8
- **Maven：** apache-maven 3.8.1-bin
- **前端环境：** Node.Js 12\14\16
